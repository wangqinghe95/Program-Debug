# 补充说明
gdb 对于 C++ 的支持
1. 断点
+ 对于 C++ 中的断点，如果设置的是函数名，则在所有的同名函数（重载）和继承的虚函数，都会设置断点，只要函数名和设置的函数一致，就都会在此处停止；如果是想指定文件或者指定类的函数，需要在函数名前加上文件或者类名的限定符
+ b +/- [number] 设置以当前运行的代码行的偏移量
+ tb [break_point] 临时断点，一次生效的
